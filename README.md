# 🛠️ プログラムで「困った」を解決します

エンジニアの **グミ** です。
プログラムの力を使って、日常の面倒な作業を自動化したり、便利なツールを作ったりしています。

### 🚀 解決できること
- **AIシステムの構築**: OpenAI APIを活用した、賢くて丁寧なAI秘書を作ります。
- **データ分析・自動化**: CSVデータなどの解析や、面倒なコピペ作業を自動化します。
- **「これ自動化できない？」という相談**: プログラムで解決できることなら何でも挑戦します。

### 💼 公開中の実績
- **[timer.html](./timer.html)**: 📚 **最新作！** 10分集中すると解放される「61秒ドボン」体内時計ゲーム付きタイマー。宿題のモチベーションを爆上げします。
- **[Urania.py](./Urania.py)**: 🔮 1日1回限定、100段階の運勢鑑定。0.1%の確率で発生する「奇跡演出」を搭載したガチ仕様占い。
- **[ai.py](./ai.py)**: 🤖 ユーザーを元気づけるビジネスAI秘書システム。丁寧な言葉遣いとエラー対応にこだわっています。
- **[stock_analyzer.py](./stock_analyzer.py)**: 📈 株価の変動を色分けして視覚的に判定する、実用的な分析ツール。

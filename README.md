# 🛠️ グミの「最強の道具箱」リポジトリ

エンジニアの **グミ** です。
「0.1%の奇跡」と「実用性」を追求し、VSコードとターミナルで開発した自作ツール群です。
**【🚀 タイトルをクリックすると各アプリが即座に起動します】**

### 💼 公開中の実績（エンジニア・グミ選定）

- **[🎰 運命の選択スロット](https://gumi21314151-max.github.io)** **【最新作】**
  夕食の献立から放課後の遊びまで、2〜4つの選択肢をセットして運命を委ねるスロット。
  **こだわりポイント:** 0.1%の確率で「全部やる」という神の宣告が下る激レア演出を搭載。迷いを断ち切る究極の意思決定ツールです。

- **[🎯 目標カウントダウン・マネージャー](https://gumi21314151-max.github.io)**
  次のテストや楽しみなイベントまでを「1秒刻み」で表示するストイックな管理ツール。
  **こだわりポイント:** localStorage（ブラウザ保存）対応。一度セットすれば、ブラウザを閉じても再読み込みしても、常にあなたの目標までの残り時間を突きつけます。

- **[🗓️ 曜日別インテリジェント・メモ帳](https://gumi21314151-max.github.io)**
  曜日ごとに背景色が自動で切り替わり、保存場所も曜日ごとに独立するスマートなメモ帳。
  **こだわりポイント:** ターミナルでローカルサーバーを立てて開発。月曜には月曜の、日曜には日曜のメモがパッと出る、実用性を極めたエンジニア仕様です。

- **[📚 61秒ドボン！集中タイマー](https://gumi21314151-max.github.io)**
  「10分間の全力宿題」の後に解放される、1分間の体内時計チキンレース。
  **こだわりポイント:** 61.00秒を超えた瞬間にドボン（強制終了）となる絶妙なスリル。宿題を「ゲームを解放するためのクエスト」に変えるモチベーション・ブースター。

- **[🔮 100段階・究極運勢鑑定](https://gumi21314151-max.github.io)**
  1日1回限定、100通りの結果から今日の運命を導き出す看板アプリ。
  **こだわりポイント:** 出現率0.1%の「虹色・激レア奇跡演出」を搭載。JavaScriptの乱数ロジックを極めた、グミ・エンジニアリングの原点です。

- **[📈 Python・株価リアルタイム分析器](./stock_analyzer.py)**
  Pandasライブラリを駆使し、株価の変動を視覚的に色分け判定する本格データ分析ツール。
  ※これはローカルのPython環境で動作させるプロ仕様のツールです。

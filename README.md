# 🛠️ グミの「最強の道具箱」リポジトリ

エンジニアの **グミ** です。
実際に自分のパソコンで動かして便利だったツールを公開しています。

### 💼 公開中の実績（エンジニア・グミ選定）

- **[memo.html](./memo.html)** 🗓️ **【曜日別メモ】** 
  曜日ごとに背景色と保存キーが自動で切り替わる、インテリジェント・メモ帳。
  ターミナルでローカルサーバーを立てて開発した、実用性重視のスマートツールです。

- **[timer.html](./timer.html)** 📚 **【集中特化】** 
  10分集中すると解放される「61秒ドボン」体内時計機能付きタイマー。
  「10分宿題 ⇄ 1分勝負」のサイクルで、勉強のモチベーションを限界まで引き出します。

- **[index.html](./index.html)** 🔮 **【看板アプリ】** 
  1日1回限定、100段階の究極運勢鑑定。
  0.1%の確率でしか発生しない「虹色・激レア奇跡演出」を搭載した、こだわりのJavaScript作品。

- **[stock_analyzer.py](./stock_analyzer.py)** 📈 **【データ分析】** 
  PythonとPandasライブラリを駆使し、株価の変動をリアルタイムで色分け判定。
  エラーを乗り越えて完成させた、視覚的に分かりやすい本格分析ツール。
